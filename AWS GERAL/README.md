# AWS
##  O que é a AWS?

A AWS (Amazon Web Services) é uma plataforma de computação em nuvem oferecida pela Amazon. Ela fornece uma ampla gama de serviços, como armazenamento, processamento, bancos de dados, redes e inteligência artificial.

Esses serviços permitem que empresas e desenvolvedores criem aplicações sem a necessidade de possuir infraestrutura física própria.


##  Infraestrutura Global

A infraestrutura da AWS é distribuída globalmente para garantir alta disponibilidade, segurança e desempenho.

### Regiões (Regions)
As regiões são áreas geográficas distintas onde os serviços da AWS estão disponíveis.

Exemplos:
- América do Sul (São Paulo)
- Estados Unidos (Virgínia do Norte)
- Europa (Irlanda)

Cada região é isolada das outras, aumentando a segurança e a redundância.


###  Zonas de Disponibilidade (Availability Zones)

Cada região possui várias Zonas de Disponibilidade (AZs), que são data centers fisicamente separados.

Características:
- Alta disponibilidade
- Tolerância a falhas
- Baixa latência entre zonas

Isso garante que, se uma zona falhar, outra pode assumir sem interromper o serviço.


## Economia da Nuvem

A AWS utiliza um modelo econômico baseado em consumo.

###  Pay-as-you-go (Pague pelo uso)

Você paga apenas pelos recursos que utiliza, sem necessidade de investimento inicial em infraestrutura.

Exemplo:
- Se usar um servidor por 2 horas, paga apenas por esse tempo
- Se armazenar 10GB, paga apenas por esse espaço


### Economia de Escala

A AWS consegue reduzir custos operacionais devido ao grande volume de clientes.

Isso permite:
- Preços mais baixos
- Infraestrutura eficiente
- Atualizações constantes

A empresa lucra com:
- Grande número de usuários
- Uso contínuo dos serviços
- Otimização de recursos em larga escala


##  Conclusão

A AWS revolucionou a forma como empresas utilizam tecnologia, oferecendo flexibilidade, escalabilidade e redução de custos.

Sua infraestrutura global e modelo de pagamento tornam a nuvem acessível tanto para pequenas empresas quanto para grandes corporações.
